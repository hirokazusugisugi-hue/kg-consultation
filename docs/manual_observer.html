1. オブザーバーとは

「オブザーバー」とは、中小企業診断士養成課程に現在在学中の学生（3期4期）が、実際の経営相談の場に同席して学ぶ制度です。

相談のスコア計算上、オブザーバーは 1点 としてカウントされます。

診断士（1期/2期） = 2点
オブザーバー（3期4期） = 1点

オブザーバーとして参加するには、事前に NDA（秘密保持誓約書）への署名 が必要です。

オブザーバーは相談の見学者であり、相談者への直接的なアドバイスは行いません。

予定が表示されない場合は、管理者がスプレッドシート上で日程を登録していない可能性があります。管理者にご確認ください。

3. NDA（秘密保持誓約書）の署名方法

NDAの署名は、オブザーバーとして相談に参加するための 必須手続き です。以下の手順に従って署名を完了してください。

  1. オブザーバー専用ページで参加予定の相談の「署名して提出」ボタンをタップ
  2. 署名フォームが表示されます:
    • 相談日: 自動入力（変更不可）
    • 相談担当者: 自動入力（変更不可）
    • 相談企業名: 自動入力（変更不可）
    • オブザーバー氏名: ご自身の名前をフルネームで入力
  3. 秘密保持誓約書の全文が表示されるので、内容を確認してください
  4. 署名欄に指またはペンで署名してください（スマートフォンの場合はタッチで直接記入）
  5. 署名を間違えた場合は「やり直し」ボタンで消去できます
  6. 内容を確認し「署名して提出」ボタンをタップ
  7. 「NDAを提出しました」と表示されれば完了です
署名は各相談日ごとに必要です。複数の相談に参加する場合は、それぞれの予定に対して署名を提出してください。
提出されたNDAはGoogle Driveに自動保存され、管理者にメール通知が届きます。特に管理者への連絡は不要です。

4. 相談見学時の注意事項

相談見学にあたり、以下の事項を遵守してください。

  • 相談開始5分前には着席・ログインしてください
  • 相談中は原則としてカメラON（Zoom）
  • メモを取ることは可能ですが、相談企業が特定できる情報の記録は禁止です
  • 録音・録画は厳禁です
  • 相談終了後、担当者から質問の機会がある場合があります
相談者が見学に不同意の場合、オブザーバーは退室する必要があります。担当者の指示に従ってください。

5. 秘密保持義務について

NDAで定められた主要条項について、わかりやすく説明します。

第1条（秘密情報の定義）

相談を通じて知り得た企業の経営情報、財務情報、技術情報、個人情報、相談資料等のすべてが「秘密情報」です。

第2条（遵守事項）

  1. 第三者への開示・漏洩をしないこと
  2. 学術研究・教育以外の目的で使用しないこと
  3. SNSやブログ等に相談企業が特定できる情報を投稿しないこと
  4. 学術目的で利用する場合は匿名化処理を行うこと
  5. 活動終了時は資料・データを返還・廃棄すること

第3条（期間および損害賠償）

秘密保持義務は活動終了後・養成課程修了後も存続します。違反時は法的責任を負います。

秘密情報の漏洩は法的責任を問われる可能性があります。相談で得た情報は一切外部に開示しないでください。
